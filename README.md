## A Proclamation

The World Wide Web was, for a brief and luminous period, **structured**.

Every region of the page occupied a cell. Every cell knew its neighbours. A document could
be drawn, in full, on a sheet of graph paper before a single character was typed. This was
not a limitation. This was **civilisation**.

Then came the great divorce — the severing of content from presentation, the exile of the
layout to a distant file ending in `.css`. We were promised flexibility. We received
**chaos**: layouts that reflow without permission, columns that collapse without warning, and
designs that look different depending on the whims of the device.

The Association exists to correct this historic error.

## What We Believe

- **The `<table>` is the rightful skeleton of every web page.** It was never merely for data.
  It was for *order*.
- **A website shall be 760 pixels wide** — on every device, in perpetuity. We call this
  *consistency*, and we regard it as a courtesy to the reader.
- **The spacer GIF is sacred.** A single transparent pixel, 43 bytes, stretched by force of
  will to hold up the entire web.
- **Cascading Style Sheets separate content from presentation.** We see no reason to separate
  two things that so plainly love each other.
- **A media query is a confession.** It admits, in writing, that the author does not know how
  wide their own website is.

## The Evidence

This repository is the Association's official website. Observe that it is built entirely in
nested tables, styled with `<font>` tags and `bgcolor` attributes, and contains **not one
byte of CSS**. It practises precisely what it preaches.

This is verifiable. Please verify it.

> *"Is this a joke?"*
> The Association has no sense of humour. This is, and has always been, a load-bearing position.
